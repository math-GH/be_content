# be_content
Contao CMS Extension. Extends view of content list

Contao CMS: [http://contao.org]

official Contao CMS extension respository: [https://contao.org/de/erweiterungsliste/view/be_content.de.html]

![](/assets/be_content.jpg)


##Settings
see System-Settings
default: title, headline, author, inColumn, tstamp, showTeaser, published, start, stop

Empty elements won't shown. 

Example elements:
news:
* tstamp (date of last change)
* headline 	
* alias 	
* author 	
* date 
* time 	
* subheadline
* teaser
* addImage
* singleSRC

calendar:
* tstamp (date of last change)
* title
* alias
* author
* startTime 
* endTime
* startDate
* endDate
* teaser
* addImage
*singleSRC
